COMANDOS DE EJECUCION - CONVERSOR GRPC

REQUISITOS:
- JDK 17 o superior.
- Maven instalado.
- Ejecutar los scripts desde PowerShell.
- No ejecutar mvn clean mientras el servidor o la GUI esten abiertos.

EJECUCION RECOMENDADA:

1. Abrir PowerShell en:
   LAB05\Propuestos\grpc_converter

2. Habilitar scripts solo para esta terminal:
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

3. Ejecutar servidor:
   .\run-server.ps1

4. Abrir otra terminal PowerShell y ejecutar GUI:
   .\run-gui.ps1

5. Demo automatica:
   .\run-demo.ps1

6. Recompilar desde cero:
   .\clean-build.ps1

COMANDOS MANUALES:

Servidor:
mvn compile exec:java "-Dexec.mainClass=com.lab.grpc.converter.ConverterServer"

GUI:
mvn compile exec:java "-Dexec.mainClass=com.lab.grpc.converter.ConverterGUI"

Demo:
mvn compile exec:java "-Dexec.mainClass=com.lab.grpc.converter.ConverterDemo"

NOTA:
Si aparecen errores como ConvertRequest, CategoryInfo o ConverterGrpc no encontrados,
cerrar todo Java, ejecutar clean-build.ps1 y volver a iniciar servidor y GUI.
